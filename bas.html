BLOKKY API 서버에 관해

BAS 소개

BLOKKY API 서버(BAS)는 제가 개인적으로 돌리고 있는 서버로, 채팅창 명령어 일부에 사용되며 여러분의 추천, 시청자 참여 요청 데이터를 저장합니다.

저장되는 정보

BAS에는 다음과 같은 정보를 저장합니다. 아래 둘은 같은 게시판 시스템을 이용하기 때문에 공통적으로 Twitch ID를 저장하게 됩니다. 이 ID는 중복(같은 시청자, 비슷한 내용) 등록 검사(!시참 사용시는 해당 X), 서버 로그에 사용되며, 외부로는 어떤 방법으로도 노출되지 않습니다(비록 이 ID는 Twitch 서비스 내에서 어렵지 않게 보실 수 있지만).

게임/콘텐츠 추천(!추천 명령어)

  • Twitch ID
  • 추천 내용

마크 시청자 참여 요청(!시참 명령어)

  • Twitch ID
  • Minecraft 사용자명(실제 Minecraft 사용자인지 검증, 같은 사용자명 중복등록 확인, 뽑기, 화이트리스트 처리를 위해)

서버 정보

서버 소프트웨어는 직접 개발하였으며, 그 외 사양은 홈페이지의 첫 페이지에 공개되어 있습니다.

서버 상태 보기

서버 정상 동작 여부는 !서버상태 명령어를 통해 보실 수 있으며, 홈페이지 왼쪽에도 있습니다.
