참고: 채널 관리자 이상만 사용 가능한 명령어입니다.

시청자 한명을 뽑습니다.

시청자들(!시참을 통해 등록한 Minecraft 사용자들)중 한명을 뽑는 명령어입니다.

사용법

!시참뽑기

출력 내용

성공한 경우

뽑았습니다! 뽑힌 시청자: Minecraft 플레이어 [사용자명]

실패한 경우

@[시청자] -> 게시판을 찾을 수 없습니다.
@[시청자] -> 함수를 찾을 수 없습니다.
@[시청자] -> 처리되지 않은 예외가 발생하였습니다.
Remote Server Returned Code [숫자]

-> 이런 메시지가 나온다면 그 경우는 보통 버그입니다.

메시지가 이상하게 나오는 경우

사실 이 경우는 버그입니다만, 아래와 같은 형식으로 메시지가 나올수도 있습니다.

@[시청자] -> {"StatusCode":404,"StatusMessage":"Board not found"}

이는 명령어 버그 혹은 서버 버그이며, 원래는 위에서 이야기한 메시지들처럼 나와야합니다.

모든 경우를 설명하기엔 너무 길어져버리니, 성공적인 경우 어떻게 나오는지만 알려드릴게요(+위에꺼는 게시판을 찾을 수 없는).

@[시청자] -> {"StatusCode":200,"StatusMessage":"Successful"}

이렇게 "StatusCode":200이 들어가있으면 정상입니다.

그렇지 않고 "StatusCode":404같은 다른게 나온다면 그건 오류입니다.
