!업타임

방송 업타임을 출력합니다.

방송 업타임(방송을 시작한지 지난 시간)을 출력합니다.

다만, 시간 출력은 영문입니다 ㅠㅠ

방송중이 아니라면 제목 대신 방송중이 아니라는 메시지가 출력됩니다.

사용법

!업타임

출력 내용

방송중인 경우

방송 업타임: [방송 업타임]

방송중이 아닌 경우

방송중이 아닙니다.

구현

$(eval
    const status = "$(twitch blokkytv "{{status}}")";
    if(status == "offline") {
        "방송중이 아닙니다.";
    } else {
        "방송 업타임: $(twitch blokkytv "{{uptimeLength}}")"; 
    } 
)
