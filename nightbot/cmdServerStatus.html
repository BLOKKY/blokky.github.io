추천 데이터 관리에 사용되는 블로키 API 서버가 잘 동작중인지 테스트하는 요청을 보냅니다.

!추천 명령의 결과로도 상태를 볼 수 있지만, 이 명령을 사용하면 추천을 등록하지 않고도 상태를 볼 수 있습니다.

정상인 경우

블로키 API 서버(BAS)는 잘 동작중입니다.

정상이 아닌 경우

Error Connecting To Remote Server

-> 보통 서버가 재부팅중이거나 아예 다운된 경우입니다.

함수를 찾을 수 없습니다.
처리되지 않은 예외가 발생하였습니다.
Remote Server Returned Code [숫자]

-> 이런 메시지가 나온다면 그 경우는 버그입니다.
