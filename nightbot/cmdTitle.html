< 이전 페이지로 돌아가기

!방제

현재 방송 제목을 출력합니다.

현재 방송정보에 설정된 제목을 출력합니다.

방송 제목이 바뀌고 나서는 새로고침을 하기 전까지는 방송 페이지에 반영되기까지 시간이 좀 걸리기 때문에, 새로고침 없이 새 제목을 보고 싶을때 유용합니다.

방송중이 아니라면 제목 대신 방송중이 아니라는 메시지가 출력됩니다.

사용법

!방제

출력 내용

방송중인 경우

현재 방송제목: [방송 제목]

방송중이 아닌 경우

방송중이 아닙니다.

구현

$(eval
    const status = "$(twitch blokkytv "{{status}}")";
    if(status == "offline") {
        "방송중이 아닙니다.";
    } else {
        "현재 방송제목: " + "$(twitch blokkytv "{{title}}")";
    }
)
